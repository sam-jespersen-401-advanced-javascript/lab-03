#### Running the app

* `npm start`
    * runs `index.js`
* `npm run lint`
    *   runs `eslint`
* `npm test`
    * runs `jest`
* `npm run test-watch`
    *   runs `jest` with the flags `--watchAll --verbose --coverage`
* `npm run jsdoc`
    * runs `jsdoc` to create doc files at `/docs/config/`

  
#### Tests
* Unit tests: `npm test`
* Lint tests: `npm run lint`
